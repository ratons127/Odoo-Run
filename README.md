# Odoo-Run
how to active and run odoo

### Step #16: switch user 'root' to 'odoo18' and active virtual directory
```
sudo su odoo18
```
```
cd /opt/odoo18
```
source odoo-venv/bin/activate
````
```
cd /opt/odoo18
```
### Step #17: Run odoo "/opt/odoo18/odoo"
```
cd /opt/odoo18/odoo
```
```
python3 odoo-bin -c debian/odoo.conf
```
